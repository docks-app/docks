Docks is for creating beautiful, live, styleguides.

A styleguide is a collection of your applicationâ€™s visual and programmatic patterns.

Docks supports patterns for JavaScript, CoffeeScript, Sass, Less, Stylus, ERB, Slim, and Haml.

## Installation

Add this line to your application's Gemfile:

    gem 'docks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install docks
